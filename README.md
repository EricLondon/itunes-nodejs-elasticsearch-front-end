## iTunes NodeJS Elasticsearch Front-end

See [this repo](https://github.com/EricLondon/ruby-import-itunes-elasticsearch) to import your iTunes XML Library into Elasticsearch.

Usage:

```
# clone repo
npm install && bower install

# start node app
node server.js

```
